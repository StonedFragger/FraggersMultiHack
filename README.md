# FraggersMultiHack
StonedFragger's Python MultiHack Tool v1 for Arma 3 and CounterStrike Global Offensive
##############################################################################################

# Prerequisites
- Arma 3 and or CSGO
- Python 3

# Instructions

# Windows:
- Create a folder and put all the files in it
- Open CMD in that folder
- start.py

# Linux:
- Create a folder and put all the files in it
- Open the terminal in that folder
- python3 start.py
